Cobalté Royale

Welcome to Cobalté Royale, a luxurious and sophisticated hotel offering world-class hospitality, fine dining, and exceptional services. Our goal is to provide an unforgettable experience for every guest.

Features

Elegant Rooms & Suites – Experience comfort with our tastefully designed accommodations.

Gourmet Dining – Indulge in exquisite meals prepared by top chefs.

Wellness & Spa – Rejuvenate with our state-of-the-art spa and wellness facilities.

Event & Conference Halls – Host memorable events and business meetings.

24/7 Concierge Service – Personalized assistance for all your needs.

Technologies Used

This project showcases the hotel's website and was built using:

HTML, CSS, JavaScript – For a responsive and interactive user experience.

Bootstrap – Ensuring a sleek and modern design.

Django – Managing backend functionalities and reservations.

PostgreSQL – Storing customer data securely.

Installation

Clone the repository:

git clone https://github.com/yourusername/cobalte-royale.git

Navigate to the project folder:

cd cobalte-royale

Install dependencies:

pip install -r requirements.txt

Run the server:

python manage.py runserver

Open your browser and visit http://localhost:8000
